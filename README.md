Naimah Farms is a **Goat Management System** designed to help small-scale farmers efficiently manage livestock records.  
This backend project is built with **Python, Django, Django REST Framework**, and uses **PostgreSQL** as the database.

---

## Features

- **Goat Management**: Add, view, update, and delete goat records.  
- **User Authentication**: Secure login/logout for farmers and admins.  
- **REST API**: Easy integration with web or mobile frontends.  
- **Admin Panel**: Manage goats and users through Django admin.  

---

## Technologies Used

- Python 3.10  
- Django 4.x  
- Django REST Framework  
- PostgreSQL  
- VS Code (IDE)  
- Git & GitHub
